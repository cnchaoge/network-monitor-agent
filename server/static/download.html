🌐 企业路由器 / 交换机 — SNMP 监控配置

如果您的设备无法安装 Agent 客户端（如企业级路由器、硬件防火墙、核心交换机），可通过 SNMP 协议接入平台监控。

📋 SNMP V2c 配置参数

SNMP 版本V2c
设备标识public
SNMP 口令public
Trap 口令public
SNMP 目的主机82.156.229.67
Trap 接收主机82.156.229.67
⚠️ 注意：任子行等部分设备仅支持 V2 版本，请选择 V2c 并确保 Community 为 public

📖 配置步骤（通用）

1. 登录路由器管理后台（通常为 192.168.1.1 或 192.168.0.1）

2. 找到「SNMP」或「网络管理」配置项

3. 启用 SNMP V2c，Community 填写 public

4. 配置 Trap/目的主机为 82.156.229.67

5. 保存设置，重启 SNMP 服务

📌 配置完成后，在管理后台添加该设备即可开始监控。
