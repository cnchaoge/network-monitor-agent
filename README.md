# 企业网络监控平台

## 功能特性

- **多网段/多目标监控**：支持配置多个目标 IP/域名，自定义监控策略
- **系统托盘**：最小化到托盘，不占用任务栏
- **开机自启**：首次运行后勾选，开机自动启动
- **多用户体系**：企业主独立账号，仅查看自己公司的网络数据
- **实时推送**：设备上报数据后，管理后台立即刷新（无需等待轮询）
- **微信报警**：网络离线超过3分钟自动推送微信通知（Server酱）
- **状态指示灯**：网关延迟/丢包率/DNS/目标可达性，颜色直观显示正常/警告/故障
- **自主注册**：企业主扫码注册，无需人工干预
- **手机端**：适配手机，扫码即看，随时随地掌握网络状态

## 快速开始

### 企业主
1. 注册账号：访问管理后台或联系管理员获取账号
2. 下载安装包：点击上方的下载链接
3. 双击运行，填写企业名称完成注册
4. 首次运行后可勾选「开机自动启动」
5. 手机访问 http://82.156.229.67:8000/mobile 登录查看

### 管理员
- 管理后台：http://82.156.229.67:8000
- 手机端：http://82.156.229.67:8000/mobile

## 技术架构

```
客户端 Agent (Windows exe)
    ↓ 每60秒上报
腾讯云服务器 (82.156.229.67:8000)
    ↓ FastAPI + SQLite
管理后台 / 手机端 / 微信报警
```
